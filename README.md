# 👶 Boardroom Kids

> Give your kid the keyboard while you work.

A colorful, noisy, joyful keyboard toy for preschoolers (ages 3–5). Every key press and screen tap explodes into big rainbow letters, musical tones, and fireworks. Zero distractions, zero instructions — just pure cause-and-effect sensory delight.

**[▶ Try it live →](https://boardroomkids.vercel.app)**

---

## What it does

- 🔤 **Big colorful letters** appear on every keypress or tap
- 🎵 **Musical tones** play in a pentatonic scale (always sounds good, even when mashed)
- 🎆 **Fireworks** burst from every tap or keypress
- 🔵 **Bouncing shapes** drift around the screen at all times
- ⛶ **Goes fullscreen** automatically on first tap — takes over the whole screen
- 📱 **Works on iPad** — touch anywhere, multi-touch supported
- 🔇 **Works offline** — no internet required after first load

## Why

Sometimes you need 10 minutes to finish a task. Your kid wants the computer. This is the answer — hand them the keyboard, open this page, and they'll be happily mashing away while you work.

## Usage

Open the URL and hand it to your kid:

```
https://boardroomkids.vercel.app
```

Or run it locally:

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```

## iPad / Add to Home Screen

On Safari: tap **Share → Add to Home Screen** for a fullscreen app icon on the home screen.

## Tech

Zero dependencies. One HTML file. No build step. No framework. No tracking. No ads.

Built with vanilla HTML, CSS, and the Web Audio API.
